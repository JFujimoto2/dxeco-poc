# SaaS台帳

## 概要
組織内で利用している全SaaSの一元管理台帳。基本情報と契約情報を管理し、利用状況を可視化する。

## URL
| 画面 | URL |
|------|-----|
| 一覧 | `GET /saases` |
| 詳細 | `GET /saases/:id` |
| 新規登録 | `GET /saases/new` |
| 編集 | `GET /saases/:id/edit` |
| CSVインポート | `POST /saases/import` |

## データ項目

### 基本情報
| 項目 | 説明 |
|------|------|
| SaaS名 | サービス名称 |
| カテゴリ | 一般IT / 不動産管理 / バックオフィス等 |
| ステータス | active / trial / cancelled |
| サービスURL | サービスのURL |
| 管理画面URL | 管理コンソールのURL |
| 契約担当者 | 社内のオーナー |
| 説明 | サービス概要 |

### 契約情報（SaasContract）
| 項目 | 説明 |
|------|------|
| プラン名 | 契約プラン |
| 月額 | 月額コスト（円） |
| 請求サイクル | monthly / yearly |
| 提供元 | ベンダー名 |
| 契約開始日 | 契約の開始日 |
| 契約満了日 | 契約の終了日 |

## 一覧画面の機能
- **キーワード検索**: SaaS名で部分一致検索
- **カテゴリフィルタ**: ドロップダウンで絞り込み
- **ステータスフィルタ**: active / trial / cancelled で絞り込み
- **CSVインポート**: CSVファイルからの一括登録
- **ページネーション**: Kaminariによるページ分割

## 詳細画面の機能
- 基本情報・契約情報の表示
- 紐づくアカウント一覧の表示
- 編集・削除ボタン

## アクセス権限
| ロール | 権限 |
|--------|------|
| admin | CRUD + CSVインポート |
| manager | CRUD |
| viewer | 閲覧のみ |
