# ダッシュボード

## 概要
ログイン後の最初の画面。SaaS管理状況の全体像を把握するためのサマリ表示。

## URL
`GET /`

## 画面構成

### KPIカード（上段）
| カード | 表示内容 |
|--------|---------|
| SaaS数 | 登録済みSaaSの総数 |
| メンバー数 | 組織内ユーザーの総数 |
| アカウント数 | 有効なSaaSアカウントの総数 |
| 要対応 | 注意が必要な項目の件数 |

### サブカード（中段）
| カード | 表示内容 | リンク先 |
|--------|---------|---------|
| 承認待ち | 未処理の申請件数 | `/approval_requests` |
| 進行中タスク | 対応中のタスク件数 | `/tasks` |
| 配信中サーベイ | アクティブなサーベイ件数 | `/surveys` |

### 最近の操作（下段・admin専用）
直近の操作ログをテーブル形式で表示。

## アクセス権限
| ロール | 権限 |
|--------|------|
| admin | 全セクション表示 |
| manager | KPI + サブカード表示 |
| viewer | KPI + サブカード表示 |
