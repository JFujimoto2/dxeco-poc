# ダッシュボード

## 概要
ログイン後の最初の画面。SaaS管理状況の全体像を把握するためのサマリ表示。

## URL
`GET /`

## 画面構成

### KPIカード（上段）
| カード | 表示内容 |
|--------|---------|
| SaaS数 | 登録済みSaaSの総数 |
| メンバー数 | 組織内ユーザーの総数 |
| アカウント数 | 有効なSaaSアカウントの総数 |
| 要対応 | 注意が必要な項目の件数 |

### サブカード（中段）
| カード | 表示内容 | リンク先 |
|--------|---------|---------|
| 承認待ち | 未処理の申請件数 | `/approval_requests` |
| 進行中タスク | 対応中のタスク件数 | `/tasks` |
| 配信中サーベイ | アクティブなサーベイ件数 | `/surveys` |

### 契約更新アラート
契約期限が近いSaaSの一覧を表示。

| 区分 | 条件 | 表示色 |
|------|------|--------|
| 期限切れ | `expires_on < 今日` | 赤（danger） |
| 7日以内 | `expires_on` が7日以内 | 赤（danger） |
| 30日以内 | `expires_on` が30日以内 | 黄（warning） |

各行にSaaS名・プラン名・期限日・残日数を表示し、SaaS詳細画面へリンク。

### コスト概要
SaaS契約コストの可視化。

| カード | 表示内容 |
|--------|---------|
| 月額コスト合計 | 全契約の月額換算合計（万円） |
| 年額コスト合計 | 全契約の年額換算合計（万円） |
| カテゴリ別コスト | Doughnut chart（Chart.js） |
| カテゴリ別内訳テーブル | カテゴリ名 + 月額コスト |

- `billing_cycle` が `yearly` の場合は月額 = 年額 / 12 で正規化
- Chart.js は importmap CDN で配信、Stimulus コントローラーで描画

### パスワード期限アラート
パスワード期限切れ・期限間近のユーザー一覧を表示。

| 区分 | 条件 |
|------|------|
| 期限切れ | `last_password_change_at` が90日以上前 |
| 期限間近 | `last_password_change_at` が76〜90日前（14日以内に期限切れ） |

### 最近の操作（admin専用）
直近の操作ログをテーブル形式で表示。

## アクセス権限
| ロール | 権限 |
|--------|------|
| admin | 全セクション表示 |
| manager | KPI + サブカード + アラート + コスト表示 |
| viewer | KPI + サブカード + アラート + コスト表示 |
