# メンバー管理

## 概要
組織内のメンバー一覧を管理する。各メンバーが保有するSaaSアカウントの一覧を確認でき、Entra IDからの同期情報を反映する。

## URL
| 画面 | URL |
|------|-----|
| 一覧 | `GET /users` |
| 詳細 | `GET /users/:id` |
| 編集 | `GET /users/:id/edit` |

※ ユーザーの作成はEntra ID同期またはdev_loginで行う（手動作成画面なし）

## データ項目
| 項目 | 説明 |
|------|------|
| 表示名 | ユーザー名 |
| メール | メールアドレス |
| 部署 | 所属部門 |
| 役職 | 職位 |
| ロール | admin / manager / viewer |
| アカウント数 | 保有SaaSアカウント数 |
| アカウント状態 | enabled / disabled（Entra IDの状態） |
| 最終ログイン | 最後にSaaS管理ツールにログインした日時 |

## 一覧画面の機能
- **名前・メール検索**: キーワードで部分一致検索
- **部署フィルタ**: ドロップダウンで絞り込み
- **ページネーション**

## 詳細画面の機能
- メンバーのプロフィール情報表示
- 保有SaaSアカウントの一覧テーブル（SaaS名、メール、ロール、ステータス）

## アクセス権限
| ロール | 権限 |
|--------|------|
| admin | 閲覧 + 編集 |
| manager | 閲覧のみ |
| viewer | 閲覧のみ |
