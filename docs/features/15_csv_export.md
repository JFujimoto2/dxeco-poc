# CSVエクスポート

## 概要
SaaS台帳、アカウント一覧、監査ログの各一覧画面から、現在のフィルタ条件を適用した状態でCSVファイルをダウンロードする。監査対応・経営報告でのデータ出力に対応。

## 対応画面

### SaaS台帳エクスポート
- **URL**: `GET /saases/export`
- **フィルタ**: キーワード（`q`）、カテゴリ（`category`）、ステータス（`status`）

| カラム | 説明 |
|--------|------|
| SaaS名 | サービス名称 |
| カテゴリ | カテゴリ |
| ステータス | active / trial / cancelled |
| URL | サービスURL |
| 担当者 | 契約担当者名 |
| プラン | 契約プラン名 |
| 月額 | 月額コスト（円） |
| 請求サイクル | monthly / yearly |
| 契約期限 | 契約満了日 |

### アカウント一覧エクスポート
- **URL**: `GET /saas_accounts/export`
- **フィルタ**: SaaS（`saas_id`）、メンバー（`user_id`）、ステータス（`status`）

| カラム | 説明 |
|--------|------|
| SaaS名 | サービス名称 |
| メンバー名 | ユーザーの表示名 |
| 部署 | ユーザーの所属部署 |
| アカウントメール | SaaSアカウントのメール |
| ロール | admin / member / owner |
| ステータス | active / suspended / deleted |
| 最終ログイン | 最終ログイン日時 |

### 監査ログエクスポート
- **URL**: `GET /admin/audit_logs/export`
- **フィルタ**: リソース種別（`resource_type`）、ユーザー（`user_id`）、日付範囲（`date_from` / `date_to`）

| カラム | 説明 |
|--------|------|
| 日時 | 操作のタイムスタンプ |
| 操作 | create / update / destroy |
| リソース種別 | 対象モデル名 |
| リソースID | 対象レコードのID |
| ユーザー | 操作者名（バッチの場合は「システム」） |
| IPアドレス | 操作元IP |

## 共通仕様
- **エンコーディング**: BOM付きUTF-8（`\uFEFF` プレフィックス、Excel互換）
- **Content-Type**: `text/csv; charset=utf-8`
- **ファイル名**: `{リソース名}_export_{日付}.csv`
- **ページネーションなし**: フィルタ条件に一致する全件を出力

## アクセス権限
| 画面 | 権限 |
|------|------|
| SaaS台帳 | 全ロール |
| アカウント一覧 | 全ロール |
| 監査ログ | adminのみ |
